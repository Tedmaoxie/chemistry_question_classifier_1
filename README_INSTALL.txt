说明：
如果运行 chemistry_backend.exe 时出现 "找不到 api-ms-win-core-path-l1-1-0.dll" 或类似的 DLL 缺失错误，
请安装微软官方的 Visual C++ Redistributable 运行库。

下载地址：
https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170

请下载并安装 "X64" 版本 (vc_redist.x64.exe)。
安装完成后，重新运行本程序即可。

此问题通常出现在 Windows 7 或未更新的旧版本 Windows 系统上。
