Chemistry Backend - 一键安装说明

1. 系统要求：
   - 推荐使用 Windows 10 或 Windows 11。
   - 不支持 Windows 7 (除非安装了 Universal C Runtime 更新 KB2533623)。
   - 如果遇到 "missing api-ms-win-core-path-l1-1-0.dll" 错误或类似的 DLL 缺失错误，说明您的系统缺少必要的运行时组件，请尝试安装 "Visual C++ Redistributable for Visual Studio 2015-2022"。去安装微软官方的 Visual C++ Redistributable 运行库。
   - 下载地址：https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170
   - 请下载并安装 "X64" 版本 (vc_redist.x64.exe)。安装完成后，重新运行本程序即可。此问题通常出现在 Windows 7 或未更新的旧版本 Windows 系统上。

2. 运行方法：
   - 双击 chemistry_backend.exe 即可启动。
   - 程序会自动打开浏览器窗口。

3. 常见问题：
   - 如果杀毒软件误报，请添加信任。
   - 如果无法连接大模型，请检查网络设置。
